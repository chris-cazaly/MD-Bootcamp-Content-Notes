Basic CMD Commands (Windows)

dir - lists contents of current directory

cd [dir] - change directory: move into named directory

cd .. - move into the directory above

code . - open current directory/file in VSCode

rm - remove directory

cls - clear screen

Basic Git Commands

git init - creates empty repo in current directory

git clone [repo] - clone repo located at [repo] onto local machine

git config --global user.name [name] - define the author name to be used for all commits

git config --global user.email [email] - definre the author email address

git add [dir/file] - stage all changes in [dir/file] for next commit

git status - list which files are stage, unstaged, and untracked

git commit -m "[message]" - commit the staged snapshot

git log - display the entire commit history

JS - Variables and Data Types

What is a variable?

A variable is used to store a value that can be accessed and/or changed later.

A variable will have 2 main features: a name and a value.

To initialise a new variable, we type: let [variable name] = [value];

The equals symbol [ = ] is known as the assignment operator, as it asigns a value to the variable.

e.g. let name = "Joe";

e.g. let age = 21;

What is a Data Type?

A data type is a classification that specifies which type of value a variable has and what it can be used for.

There are lots of different data types and we will look at some of them below, starting with the number type

Numbers

In JavaScript, a number is any numerical value. It can be an integer or a decimal. It can be positive or negative.

We can perform numerical functions with number types, such addition, subtraction, multiplition, and division.

There are more complex numerical functions that can be applied too.

e.g. let age = 26;

e.g. let price = 4.99;

e.g. let result = -10.5;

You can add numbers together explicitly:

e.g. let result = 2 + 4; [result = 6]

Or you can add two variables together, provided they are both number types:

e.g.

let a = 2;

let b = 4;

let result = a + b; [result = 6]

Strings

A string is a series of characters.

A string can be any text inside double [ " ] or single [ ' ] quotes.

e.g. let name = "Joe";

e.g. let inspirationalQuote = "Don't take life too seriously."

Strings can also be "added" together. This is known as concatenation:

e.g. let name = "Joe" + "Bloggs"

Boolean

A binary variable can only have of two possible values, true or false.

e.g. let isRaining = true;

e.g. let isEnabled = false;
